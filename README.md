# Lumen Tidy

A standalone Astro landing-page product for a residential and commercial
cleaning company. The page is a Webdevful-owned adaptation of the Tidex section
topology, built with original copy, local Pexels photography, local Fontsource
typography, and a WLFS-generated identity.

## Commands

```sh
npm install
npm run design:lint
npm run build
npm run preview
```

## Evidence

- `BUILD-CONTRACT.json` records the P4/P5 source-port contract.
- `docs/ASSET-SOURCES.md` records approved media provenance.
- `qa/lpf-visual-fidelity/lumen-tidy/` contains source/output pairs and gate results.
- `qa/lighthouse-mobile.json` and `qa/lighthouse-desktop.json` contain performance evidence.

Public demo: <https://lumen-tidy-landing-page.pages.dev/>
